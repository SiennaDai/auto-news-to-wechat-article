# 微信公众号文章 HTML 生成规范

## 一、HTML 标签规范（必须遵守）

### 文档结构
- 所有内容用一个根 `<div>` 包裹：开头 `<div>`、结尾 `</div>`
- 不得输出 `<html>`、`<body>`、`<head>` 等外层标签

### 标题
- 使用 `<h1>`、`<h2>`、`<h3>` 标签
- 禁止使用 `<h4>`-`<h6>`（会被自动降级）
- 全文只有一个 `<h1>`

### 段落
- 每个自然段用 `<p>` 标签包裹
- 每个 `<p>` 必须有首行缩进：`style="text-indent: 2em;"`
- 段落间不留空行，`</p><p>` 直接相连
- `<div>` 内的裸文本也需要用 `<p>` 包裹

### 重点标记
- 使用 `<strong>` 标记，禁止 `<b>`（会被自动替换）
- 每段 `<strong>` 数量：**不少于 2 处，不超过 5 处**
- `<strong>` 不可嵌套

### 禁止项
- 禁止 Markdown 语法：`#`、`##`、`**`、`*` 等
- 禁止 `<i>` 标签，用 `<em>` 代替
- 禁止在非 `<p>` 元素中直接放置文本

## 二、图片占位格式（必须遵守）

封面图放在 `<h1>` 下方，正文图片放在对应段落后，尾图放在最后。

```html
<div class="image-placeholder" data-images="描述.jpg" data-layout="single"></div>
```

| 图片数 | data-layout | 分隔符 |
|--------|-------------|--------|
| 1 张 | `single` | 无 |
| 2 张 | `side-by-side` | `|` |
| 3 张+ | `carousel` | `|` |

多张图片时将文件名用 `|` 连接：`data-images="图1.jpg|图2.jpg"`

## 三、输出要求（必须遵守）

- 只输出 HTML，不得输出任何解释、说明、分析文字
- 输出必须以 `<div>` 开头、以 `</div>` 结尾
